API Documentation

Welcome to the API documentation for the PySpamDetector package.

This package allows you to classify text as spam or not spam and provides a description of the text content. Below are the available endpoints and usage instructions:

Endpoints

  • POST /read_text: Classify text as Spam or Not Spam.
  • POST /describe_text: Get a description of the text.

Usage

To use the PySpamDetector package, first install it using pip:

pip install pyspamdetector

Once installed, you can use the package in your Python scripts as follows:


from pyspamdetector import PySpamDetector

detector = PySpamDetector()

# Classify text
result = detector.read_text("Your text here")
print("Is spam:", result)

# Describe text
description = detector.describe_text("Your text here")
print("Description:", description)
